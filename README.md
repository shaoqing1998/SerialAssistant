# SerialAssistant - 串口调试助手

这是一个纯AI项目
一个基于 **Python + PySide6 + pyserial** 的串口调试工具，支持数据过滤和关键词高亮（规划中）。

## 功能特性（v1.0）

- ✅ 自动扫描并列出可用串口
- ✅ 支持常用波特率（1200 ~ 921600）
- ✅ 后台线程读取串口数据，不阻塞 UI
- ✅ 接收日志区（等宽字体，深色主题）
- ✅ 支持文本 / HEX 两种显示模式
- ✅ 自动滚动到最新数据
- ✅ 发送区支持文本 / HEX 两种发送模式
- ✅ 支持 Ctrl+Enter 快速发送
- ✅ 可选追加 `\r\n` 换行符
- ✅ 底部状态栏显示收发字节计数
- ✅ 配置自动保存/加载（config.json）

## 规划功能（后续版本）

- 🔲 多 Tab 数据过滤
- 🔲 关键词高亮（自定义颜色）
- 🔲 正则表达式过滤
- 🔲 数据录制与回放
- 🔲 时间戳显示

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行程序

```bash
python main.py
```

## 项目结构

```
SerialAssistant/
├── main.py              # 主入口和主窗口 UI
├── serial_manager.py    # 串口连接管理（pyserial 封装）
├── log_viewer.py        # 日志显示组件（支持高亮和过滤）
├── filter_manager.py    # 过滤器管理（多 Tab 过滤，规划中）
├── config.py            # 配置文件加载/保存
├── config.json          # 默认配置
├── requirements.txt     # 依赖包
└── README.md            # 本文件
```

## 配置文件说明（config.json）

| 字段 | 说明 | 默认值 |
|------|------|--------|
| `serial.port` | 上次使用的串口 | `""` |
| `serial.baudrate` | 波特率 | `115200` |
| `ui.window_width` | 窗口宽度 | `1100` |
| `ui.window_height` | 窗口高度 | `650` |
| `ui.font_family` | 日志字体 | `"Consolas"` |
| `ui.font_size` | 日志字号 | `10` |
| `ui.max_log_lines` | 最大日志行数 | `5000` |
| `send.add_newline` | 发送时追加换行 | `true` |
